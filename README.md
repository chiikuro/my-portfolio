# CMS不使用ポートフォリオサイト

## 概要
HTML / CSS / JavaScript を使って作成したWebサイトです。
CMS（WordPress等）を使わず、1からコーディングしています。

## 使用技術
- HTML
- CSS
- JavaScript

## 工夫した点
- レスポンシブ対応でスマホでも閲覧しやすくしました。
- デザインと利便性の両方を意識しました。
- 営業状態が一目で分かり、とホバーで開店・閉店までの時間が分かるように工夫しました。
